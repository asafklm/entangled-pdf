Why is this needed?

The token ensures that only you can trigger inverse search. The token was printed when you started the server with entangle-pdf start. You can also run entangle-pdf status to see the current token.

Cross-Device Access

If you're accessing this from a different device, use a clipboard syncing mechanism (like iCloud, KDE Connect, or similar) to copy the token from your terminal to this browser.
